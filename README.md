# 🎓 راهنمای جامع دانشجویی دانشگاه یزد (نسخه وب)

این پروژه یک وب‌سایت استاتیک و کاربردی برای راهنمایی دانشجویان جدیدالورود و فعلی دانشگاه یزد است که اطلاعات مربوط به بخش‌های مختلف دانشگاه را به صورت دسته‌بندی شده ارائه می‌دهد.

## 🚀 ویژگی‌ها
- **بخش سلف سرویس:** راهنمای رزرو غذا، معرفی سلف‌های آفتاب، مهتاب، غدیر و ترنج.
- **بخش خوابگاه:** راهنمای اسکان و قوانین خوابگاهی.
- **کتابخانه و امکانات:** معرفی کتابخانه مرکزی و سایر امکانات رفاهی.
- **طراحی ریسپانسیو:** بهینه‌شده برای نمایش در موبایل و دسکتاپ.
- **رابط کاربری ساده:** استفاده از سیستم Grid برای دسترسی سریع به منوها.

## 🛠 تکنولوژی‌های استفاده شده
- HTML5
- CSS3 (Custom Properties & Grid)
- JavaScript (Vanilla JS)

## 💻 نحوه اجرا
برای مشاهده پروژه به صورت محلی:
1. فایل‌ها را دانلود (Clone) کنید.
2. فایل `index.html` را در مرورگر خود باز کنید.
